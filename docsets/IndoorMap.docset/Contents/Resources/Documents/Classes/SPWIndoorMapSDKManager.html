  • initialSDK(slug:shops:language:appDelegate:)

    This is method for initialize IndoormapSDK this method contains 3 steps 1. set departmentSlug, 2. set SDK language, 3. initialize api service for Mappedin SPWIndoorMapSDKManager.shared.initialSDK(shops: [], appDelegate: AppDelegate.self)

    • slug: slug is the department name that’s an admin set on mappedin backend e.g. siam-paragon.
    • shops: shops is an array of Dictionary Object that’s come from AIS
    • language: language is a current active language
    • appDelegate: for initialize Mappedin API Service you need to init with UIApplicationDelegate.Type
